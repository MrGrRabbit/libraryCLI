
# Electronic Library #

Название системы — электронная библиотека.
***  <!-- !!!! убрать -->

## Description ##

Электронная библиотека предназначена для:

* хранения книг;
* добавления книг;
* просмотра книг;
* удаления книг
* контроль пользователя администратором.
